## 강의소개

- 스프링을 왜 공부해야하는가
  - 실무에서 잘 동작하는 웹 어플리케이션을 개발하기 위해

- 간단한 웹 앱을 개발할 예정
  - 플젝 생성
  - 부트로 웹 서버 실행
  - 회원 도메인 개발
  - 웹 MVC 개발
  - DB 연동 - JDBC, JPA, 스프링 데이터 JPA
  - 테스트 케이스 생성

- 강의 목표
  - `스프링 학습의 제대로 된 첫 길잡이 역할`

- 다루지 않는 것
  - 스프링 기술 그자체에 매몰 X
  - 오래된 스프링 기술 X
  - 마이너한 스프링 기술 X

- 학습 방법
  - 처음부터 끝까지 직접 코딩

- 스프링 완전 정복 로드맵
  - 스프링 입문 (지금 여기!)
  - 스프링 핵심 원리
  - 스프링 웹 MVC
  - 스프링 DB 데이터 접근 기술
  - 실전! 스프링 부트

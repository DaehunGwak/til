01. 들어가기

해당 내용은 이펙티브 자바 3판 을 읽고 정리한 글이다.

책의 규칙#

  • 핵심 규칙
    • 명료성
    • 단순성
  • 컴포넌트는 사용자를 놀라게하는 동작 X
  • 코드는 복사보단 재사용
  • 의존성은 최소로
  • 오류는 만들어지자마자 가능한 한 빨리 잡아야함

책의 중점#

  • 성능에 집중하는 파트는 많지 않음, 다음과 같은 부분에서 많이 다루게 됨
    • 명확
    • 정확
    • 유용
    • 견고
    • 유연
    • 관리용이

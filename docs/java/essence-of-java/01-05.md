## ch04. 조건문과 반복문

- 반복문에 이름을 붙여 중첩 안쪽에 있는 break로 밖의 반복문을 탈출할 수 있음

```java
Loop1 :
for (int i = 0; i <= 100; i++) {
    for (int j = 0; j <= 300; j++) {
        if (j > 100)
            break Loop1; // Loop1 으로 명명된 for (int i...) 부분 탈출
    }
}
```

## ch05. 배열

- JVM이 모든 배열의 길이를 별도로 관리하고 있음
- for문으로 배열카피 하는 것 보단 `System.arraycopy()` 를 사용하는 것이 효율 적
  - 값을 지정된 길이만큼 통으로 복사
